# Data-Exploration-Preparation-CA1---Salary-Survey-Dataset-
## Project Overview
This project was developed as part of the Continuous Assessment for the 
Data Analytics for Business programme at CCT College Dublin.

The main objective of this project is to explore, clean, and prepare raw datasets
for further analysis and modelling, as well as to extract meaningful insights
using Python.

## Dataset Description
The dataset contains structured business-related data used for analytical purposes.
It includes numerical and categorical variables that require preprocessing
before analysis.

Key features of the dataset include:
- Financial and operational indicators
- Customer and transaction-related attributes
- Time-based variables

## Methods and Tools
The following techniques and tools were used in this project:

- Data cleaning and preprocessing:
  - Handling missing values
  - Removing duplicates
  - Correcting data formats
  - Validating data types

- Exploratory Data Analysis (EDA):
  - Descriptive statistics
  - Distribution analysis
  - Correlation analysis
  - Outlier detection

- Data Visualisation:
  - Line charts
  - Bar charts
  - Histograms
  - Scatter plots

Tools and libraries:
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Results and Insights
The analysis identified key patterns and relationships within the dataset, including:

- Trends and seasonality in selected variables
- Correlations between key performance indicators
- Potential data quality issues affecting analysis
- Areas for further modelling and prediction

These insights can support data-driven decision-making in business environments.
